Une phrase entière se compose de deux parties.

Il s'agit de :

  1. Sujet
  2. Prédicat

Le sujet est celui dont la phrase parle

Et le prédicat est la description principale du sujet.

Il existe trois genres de prédicats.
Il s'agit de :

  1. Prédicat nominal
  2. Prédicat adjectival
  3. Prédicat verbal

Par exemple :

  • Tanaka-san (est étudiant).
  • Le chien (est mignon).
  • Le tournesol (se tourne).

Prédicat nominal

で　あります

Prédicat adjectival

Idem comme dans prédicat nominal, sauf que, il existe un groupe d'adjectifs qui se conjuguent soit-même.

Il s'agit de い.

Il se terminent toujours on i. Mais fait attention ! Pas tous les adjectifs qui se terminent en -i sont des adjectifs en い.
