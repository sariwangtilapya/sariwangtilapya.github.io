The pronoun "one"

  1. The pronoun when there is only the need to add a description.
    For example:
    1. Could you pass me the ball? The blue one.
    2. I'm gonna buy a lizard - the one that eats cockroaches.

    There is no pronoun "one" in Tagalog. One just needs to treat the description as a nominal group itself and decline it appropriately.

    1. Pakiabot nga¡ nung bola. Yung blue.
    2. Bībilí ako ng butikí¡ - yung naŋgaŋgain ng ipis.
  2. The non-second person/general "you" / you / one / man
    For example:
    1. Can you/one last underwater for five minutes?

    None of this either. Use an impersonal subject.

    1. Pwede bang tumagál ng limáng minuto sa ilalim ng tubig?
      Is it possible to last underwater for five minutes?
