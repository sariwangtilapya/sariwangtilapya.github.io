Saying yes or no

The words for yes and no are:

  • yes - oo
  • no - hindí¡

However, that is not always the case. There are questions to which you can't answer with OO or HINDÍ¡.

If the question asks if one has or there is, it must be answered with:

  • meron - there is / one has
  • walá¡ - there isn't / one hasn't

There are questions to which you can elect to answer with OO or HINDÍ¡ or be emphatic with:

If the question asks if something is permitted or not, to be emphatic, it can be answered with:

  • pwede - It is permitted / possible
  • bawal - It is prohibited / impossible

If the question asks if one wants or not, to be emphatic, it can be answered with:

  • gustó - One wants it / to.
  • ayaw - One doesn't want it / to.
  • ayoko - It don't want it / to.
